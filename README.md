# Customer-personal-Analysis
## 📌 Project Overview
This project focuses on analyzing customer data to understand different customer personalities, behaviors, and purchasing patterns. The goal is to help businesses make better marketing decisions based on customer segmentation.

---

## 🎯 Objectives
- Analyze customer demographics and behavior
- Identify patterns in customer spending
- Segment customers into different groups
- Provide insights for targeted marketing

---

## 📊 Dataset
- File: campaign.csv
- Contains customer information like:
  - Age
  - Income
  - Education
  - Spending habits
  - Campaign response

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 🔍 Exploratory Data Analysis (EDA)
- Handling missing values
- Data cleaning
- Feature engineering
- Data visualization
- Correlation analysis

---

## 📈 Key Insights
- Identified high-value customers
- Observed spending patterns across age groups
- Analyzed campaign success rates
- Segmented customers based on behavior
